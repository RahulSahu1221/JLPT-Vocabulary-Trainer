## ✨ Features
* **🧠 Spaced Repetition System (SRS):** Smart flashcards that mathematically calculate when you need to review a word before you forget it, optimizing your study time.
* **⚡ Interactive Quizzes:** Test your knowledge with Kanji-to-Meaning and Meaning-to-Kanji modes to ensure full comprehension.
* **📱 Fully Offline (PWA):** Installs directly to your phone or desktop. The Service Worker architecture ensures it works perfectly without an internet connection.
* **🎨 Spatial Frosted-Glass UI:** Modern, minimalist aesthetic featuring a custom Vanilla WebGL "LightRays" background.
* **📊 Advanced Analytics:** Track your daily goals, current streaks, and overall learning progress on a comprehensive dashboard.
* **⌨️ Command Palette:** Press `Ctrl + K` to quickly navigate the app, search vocabulary, or toggle themes.

## 🛠️ Tech Stack
* **Frontend:** Vanilla JavaScript, HTML5, CSS3 
* **Graphics:** WebGL (Vanilla)
* **Architecture:** Progressive Web App (PWA) with Service Workers & Manifest
* **Data:** LocalStorage API & JSON fetch architecture
